# Website

Vue 3 + Vite portfolio site scaffolded for GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

Update `base` in `vite.config.js` to match your GitHub repository name:

```js
base: '/your-repo-name/'
```

Then run:

```bash
npm run deploy
```
